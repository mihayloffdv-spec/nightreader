# Reader Mode Evolution — Design Doc

Generated by /plan-ceo-review on 2026-03-17
Branch: main | Mode: SCOPE EXPANSION

## Vision

**Goal:** Make Reader Mode feel like reading an ePub in Apple Books — not a PDF viewer with text extraction.

**The feeling:** "I forgot I was reading a PDF."

## Accepted Scope

### 1. Theme-Aware Snapshot Rendering
When reading in a dark theme (Midnight, Ember, etc.), table/diagram snapshots currently appear as white blocks. Apply CIColorInvert + warm tint to snapshot UIImages based on the active theme.

### 2. Cross-Page Paragraph Joining
When a page ends mid-sentence and the next page continues it, join them into one continuous paragraph. Remove the artificial page divider between them. Detection: last text block doesn't end with sentence punctuation AND next page's first text block starts lowercase.

### 3. Smooth Scroll Position Persistence
Save and restore the ScrollView Y offset within Reader Mode, not just the page index. Users land exactly where they left off when reopening.

### 4. Estimated Reading Time
Show "~X min remaining" in the bottom bar. Calculate total word count upfront using page.string on document open (~50ms for 200 pages). Use ~200 words/min for Russian text.

### 5. Quick A-/A+ Font Size Buttons
Add A- and A+ buttons to the top toolbar (visible only in Reader Mode). Single tap adjusts font size by 2pt. Eliminates the need to open Display Settings for the most common adjustment.

### 6. Refactor PDFContentExtractor
Split the 2,266-line God object into 4 focused modules:
- **PDFContentExtractor** (orchestrator, ~200 LOC) — coordinates extraction pipeline
- **TextExtractor** — paragraph splitting, dehyphenation, drop cap recovery, OCR fallback
- **ImageExtractor** — CGPDFScanner, XObject decode, image extraction
- **LayoutAnalyzer** — multi-column detection, gap detection, heading detection
- **PageRenderer** — region render, full page render, theme-aware CIFilter

Shared types (ContentBlock, ExtractedImage, BlockCache) go in ContentTypes.swift.

## Architecture

```
ReaderView → ReaderViewModel → PDFContentExtractor (orchestrator)
                                  ├─▶ TextExtractor
                                  ├─▶ ImageExtractor
                                  ├─▶ LayoutAnalyzer
                                  └─▶ PageRenderer (+ theme filter)
                                         ▼
                                    BlockCache (+ word count)
```

### Cross-Page Joining Flow
```
Page N last block         Page N+1 first block
       │                         │
       ▼                         ▼
 Ends without ".!?»"    Starts lowercase?
       │                         │
       YES + YES ──▶ Join into single .text, remove divider
       Otherwise  ──▶ Keep page divider
```
