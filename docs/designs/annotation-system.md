## Vision

### 10x Check
Transform NightReader from a PDF viewer into a personal knowledge operating system:
- Living knowledge graph connecting highlights across books
- Spaced repetition built into the reading flow
- AI argument maps (thesis → evidence → conclusion)
- Actions become reminders with progress tracking in library

### Platonic Ideal
The perfect reading companion feels like a brilliant friend who read the book before you. Light pencil marks next to important sentences. One perfect question after each chapter. Distilled takeaways after the book. Gentle reminders of commitments weeks later.

Zero friction during reading (passive AI), maximum depth when engaged (active annotation), lasting impact after the book (export + spaced recall).

### Error Handling Requirements
1. Disk full on JSON save → catch, show alert
2. Corrupted JSON → log, start fresh with backup
3. AI malformed response → parse defensively, fall back gracefully
4. AI model refusal → detect, show friendly "AI unavailable" message
5. Concurrent writes → serial DispatchQueue with debounce (existing pattern)
